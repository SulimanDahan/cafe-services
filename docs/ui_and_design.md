# 🎨 واجهة المستخدم ونظام التصميم (UI & Design)

تم بناء الواجهة بناءً على معايير الجمالية العالية المستوحاة من **Material You**، مع الاستفادة الكاملة من إمكانيات **Tailwind CSS v4**.

## 1. التنسيقات المركزية (Global Tokens)
تطبيقنا يستخدم أسلوب المتغيرات المركزية داخل `src/app/globals.css`.
يمكنك تغيير لون الهوية بالكامل أو تعديل الزوايا بتغيير المتغيرات فقط، وسينعكس ذلك على النظام كله.

```css
:root {
  /* ألوان النظام الأساسية */
  --color-brand: #f59e0b;       /* اللون الذهبي الأساسي */
  --color-page-bg: #07080a;     /* خلفية الصفحات الداكنة */
  --color-surface: #131522;     /* خلفية البطاقات والحاويات */

  /* أنصاف الأقطار */
  --radius-card: 28px;
  --radius-modal: 24px;
}
```

## 2. المكونات الأساسية (Components)
تم تجريد عناصر الواجهة إلى مكونات قابلة لإعادة الاستخدام داخل `src/components`:
- **`PrimaryButton`**: زر أساسي يدعم الحركة والتفاعل اللوني.
- **`MetricCard`**: بطاقة لعرض الإحصائيات مع رموز تعبيرية وخلفيات زجاجية (Glassmorphism).
- **`NotificationCenter`**: مركز إشعارات تفاعلي يقرأ البيانات من الـ SSE ويحدث الواجهة تلقائياً.

## 3. تأثيرات Glassmorphism
النظام يتبنى التصميم الزجاجي باستخدام `backdrop-blur` والشفافية. عند الحاجة لتمييز العناصر العائمة (Modals, Overlays)، نستخدم ألوان شفافة بحدود خفيفة:
`bg-overlay/80 backdrop-blur-xl border border-white/10`

## 4. تعدد اللغات (i18n)
واجهة المستخدم مدعومة باللغتين العربية والإنجليزية. يتم جلب النصوص من مجلد `src/translations/` باستخدام الـ Hook:
```typescript
import { useLanguage } from "@/config/i18n";

const { t, isRtl } = useLanguage();
```
ويدعم النظام التبديل الفوري بين الاتجاهات (RTL / LTR) مع الرسوميات والتخطيطات المتوافقة.
