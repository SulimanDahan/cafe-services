# ☕ Cafe Services Dashboard

**نظام الإدارة الرقمية الذكي للمقاهي والفنادق** 🚀  
مشروع تقني متكامل مبني باستخدام أحدث تقنيات الويب (**Next.js 16, Prisma ORM, PostgreSQL**) يوفر تجربة سلسة وعالية الأداء لإدارة الطلبات، الحجوزات، والعملاء.

يتميز النظام بواجهة استخدام فائقة الجمال مصممة بنمط **Glassmorphism** وتدعم الوضع الداكن (Dark Mode) بشكل أساسي، بالإضافة إلى نظام إشعارات لحظية (Real-Time) متطور ومستقل بالكامل بدون الاعتماد على خدمات خارجية.

---

## ✨ أبرز مميزات النظام (Highlights)

- **⚡ إشعارات لحظية مدمجة (Native SSE):** استقبال الطلبات والحجوزات الجديدة مباشرة دون إعادة تحميل الصفحة، باستخدام تقنية `Server-Sent Events` الخفيفة والسريعة.
- **🎵 تنبيهات صوتية ذكية (Web Audio API):** توليد نغمات تنبيه برمجية داخل المتصفح عند وصول إشعار جديد دون الحاجة لتحميل ملفات صوتية ثقيلة.
- **🐳 بيئة معزولة (Dockerized):** المشروع مهيأ بالكامل للعمل داخل حاويات (Containers) لضمان استقراره التام وسهولة نقله بين بيئات التطوير (Dev) والإنتاج (Prod).
- **🎨 تصميم مركزي (Centralized UI):** نظام تصميم مرن مبني بـ `Tailwind CSS v4` يتيح تغيير الألوان وتدوير الحواف في جميع شاشات النظام من خلال ملف CSS واحد.
- **🌍 متعدد اللغات (i18n):** دعم كامل للغتين العربية والإنجليزية مع التبديل الفوري لاتجاه الشاشة (RTL / LTR).
- **🔒 أداء وقوة قاعدة البيانات:** ربط آمن وسريع باستخدام **Prisma ORM** مع **PostgreSQL** للتعامل مع آلاف السجلات بكفاءة، مع حفظ البيانات والصور بفضل `Docker Volumes`.

## 📖 التوثيق الشامل (Comprehensive Documentation)

نظراً لحجم المشروع وغناه بالتقنيات الحديثة، تم إعداد مكتبة توثيق متكاملة تشرح أدق تفاصيل النظام لمساعدتك على فهمه وتطويره:

1. **[📂 هيكلية المشروع ومجلداته (Project Structure)](docs/structure.md)**
   شرح تفصيلي لأماكن الملفات، مسارات الـ API، أين تجد المكونات، وكيفية تنظيم الكود المصدري.

2. **[⚙️ معمارية النظام والتقنيات (Architecture & Real-Time)](docs/architecture.md)**
   كيف يعمل محرك الإشعارات اللحظي؟ وما هي التقنيات الأساسية (Tech Stack) التي يقوم عليها المشروع؟

3. **[🚀 إعدادات التشغيل والديبلويمينت (Docker & Deployment)](docs/deployment.md)**
   دليل خطوة بخطوة لتشغيل المشروع في بيئة التطوير (Development) أو رفعه على الخادم (Production) بأعلى أداء.

4. **[💾 قواعد البيانات (Prisma & PostgreSQL)](docs/database.md)**
   تعرف على هيكل جداول قاعدة البيانات وكيفية مزامنتها بطريقة تمنع فقدان البيانات.

5. **[🎨 واجهة المستخدم ونظام التصميم (UI, Design & CSS Variables)](docs/ui_and_design.md)**
   كيف يعمل التصميم الزجاجي، وكيف يمكنك تغيير الألوان الرئيسية (Theme) في ثوانٍ.

> **نظرة سريعة للبدء (Quick Start):**
> لتشغيل المشروع محلياً وبشكل فوري، تأكد من تثبيت Docker ثم نفذ الأمر التالي في سطر الأوامر:
>
> ```bash
> docker compose -f docker-compose.dev.yml up --build
> ```
>
> سيرفر التطوير سيكون متاحاً على الرابط: `http://localhost:3000`
